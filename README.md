# 💰 Conversor de moedas

Este é um projeto de conversor de moedas desenvolvido durante os estudos de Full Stack Web Development. A aplicação permite converter valores entre diversas moedas em tempo real, utilizando uma API externa para garantir que as taxas de câmbio estejam sempre atualizadas.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

* **HTML5** (Estrutura)
* **CSS3** (Estilização com Glassmorphism)
* **JavaScript** (Lógica de conversão e consumo de API)
* **AwesomeAPI** (Integração de dados financeiros em tempo real)

---

## 🛠️ Funcionalidades

* **Conversão Inteligente:** Converte entre Real, Dólar, Euro, Libra, Iene e Bitcoin.
* **Lógica "De-Para":** Diferente de conversores simples, você pode escolher a moeda de origem e a de destino livremente.
* **Dados em Tempo Real:** As taxas não são fixas no código; elas são buscadas no mercado financeiro no momento do clique.
* **Design Responsivo:** Adaptado para visualização em computadores e dispositivos móveis.
* **Interface Moderna:** Estética Dark Mode com efeitos de desfoque e brilho neon.
